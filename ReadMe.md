# 古籍對勘與版本分析工具

主要針對地方志的需求設計，其他古籍亦適用。

**使用方法：**

- 本地運行

  在代碼所在文件夾路徑中打開命令窗口，運行`streamlit run text_review_and_version_analysis.py`

- streamlit 免費服務器運行

  打開網址：<https://textreviewandversionanalysis-sszacedfatmtftzzuc9vss.streamlit.app/>
